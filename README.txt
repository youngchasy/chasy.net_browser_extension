chasy.net extension

Что внутри:
- профили прокси (HTTP / HTTPS / SOCKS5)
- светлая и тёмная тема
- импорт / экспорт профилей и настроек
- понятный статус подключения
- автотест после включения (внешний IP и примерный ping)
- шифрование логина и пароля через пароль хранилища
- базовые правила маршрутизации по домену и IP/CIDR

Установка:
1) зайди в расширения браузера
2) нажми "Загрузить распакованное"
3) выбери папку расширения

Примечания:
- SOCKS5-аутентификация в Chromium-браузерах работает нестабильно.
- Ping и внешний IP считаются best-effort через сетевые запросы расширения.
- Geo-rules не включены: для них нужен отдельный geo-IP источник или серверная база диапазонов.
- Статистика ↓/↑ приблизительная, по Content-Length.
