chasy.net extension

RU:

Что внутри:
- профили прокси (HTTP / HTTPS / SOCKS5)
- светлая и тёмная тема
- импорт / экспорт профилей и настроек
- понятный статус подключения
- автотест после включения (внешний IP и примерный ping)
- шифрование логина и пароля через пароль хранилища
- базовые правила маршрутизации по домену и IP/CIDR

Установка:
1) зайди в расширения браузера
2) нажми "Загрузить распакованное"
3) выбери папку расширения

Примечания:
- SOCKS5-аутентификация в Chromium-браузерах работает нестабильно.
- Ping и внешний IP считаются best-effort через сетевые запросы расширения.
- Geo-rules не включены: для них нужен отдельный geo-IP источник или серверная база диапазонов.
- Статистика ↓/↑ приблизительная, по Content-Length.

EN:

What's inside:
- Proxy profiles (HTTP/HTTPS/SOCKS5)
- Light and dark themes
- Import/export profiles and settings
- Clear connection status
- Autotest after enablement (external IP and approximate ping)
- Login encryption and password storage entry
- Basic routing rules by domain and IP/CIDR

Installation:
1) Open the browser extension
2) Click "Load unpacked"
3) Select the extension

Notes:
- SOCKS5 authentication in Chromium browsers is standalone.
- Ping and external IP addresses are received with maximum efficiency through the network request extension.
- Geo-rules are not enabled: they require a separate geo-IP source or server database range.
- Appropriate ↓/↑ statistics, based on Content-Length.
